# TuPrimeraPagina-Cerpa
Primera página web para TP3

Este proyecto consiste en una aplicación web para la gestión de un centro de masajes.

# Modelos creados
Cree tres modelos: cliente, servicio (refiere a tipo de masajes), turno

# Funcionalidades de la página web
En la página web se puede observar:
- Página de inicio
- Búsqueda de clientes por nombre
- Creación de clientes
- Creación de servicios
- Creación de turnos

# Orden para probar la aplicacion
1. Clonar el repositorio
2. Crear y activar un entorno virtual
3. Instalar Django
4. Ejecutar migraciones
5. Crear un superusuario
6. Ingresar al panel de administración
7. Probar las funcionalidades desde la web

# Pasos para ejecutar el proyecto
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
